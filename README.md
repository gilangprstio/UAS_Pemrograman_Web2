# Toko Bangunan Pro

Aplikasi web berbasis Flask untuk mengelola inventaris dan transaksi penjualan toko bangunan.

## Deskripsi
Sistem ini dirancang untuk mempermudah admin toko bangunan dalam mengelola stok barang secara digital dan mencatat setiap transaksi penjualan dengan cepat.

## Fitur Utama
- **CRUD Barang:** Menambah, menampilkan, dan menghapus data barang.
- **Manajemen Stok:** Memantau ketersediaan barang secara real-time.
- **Sistem Transaksi:** Mencatat penjualan barang.

## Teknologi yang Digunakan
- Python
- Flask Framework
- SQLite (Database)
- HTML/CSS

### Panduan Menjalankan Aplikasi
1. Pastikan Python sudah terinstal.
2. Buka terminal di folder proyek ini.
3. Instal library yang dibutuhkan: `pip install flask flask-sqlalchemy`
4. Jalankan aplikasi dengan perintah: `python app.py`
5. Buka browser dan akses alamat: `http://127.0.0.1:5000/`
